# Local Live Translate

An iPhone-friendly live subtitle / translation web app.

## Repository structure

- `docs/` — deployable GitHub Pages frontend
- root experimental files — local prototype and research artifacts

## GitHub Pages

Enable GitHub Pages and set the source to:

- **Deploy from branch**
- Branch: `main`
- Folder: `/docs`

Then the app will be available at:

- `https://<your-github-username>.github.io/<repo-name>/`

## Current deployment recommendation

Use GitHub as the source of truth and GitHub Pages as the first stable public frontend.

## Notes

The old local prototype used device-local experimentation paths. Those are kept for research, but the public deploy target should be the `docs/` frontend.
